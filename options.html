Notes

  • PAC stands for Proxy Auto-Configuration. Under the hood, this extension automatically generates and installs a PAC script.
  • shexp patterns are wildcard patterns.
  • regex patterns are JavaScript regular expressions.
  • When matching against https URLs, the path and query components are stripped as described here.
  • ProxyAlien starts in Unmanaged mode, in which it does not control proxy settings.
  • If another extension or script alters proxy settings, ProxyAlien displays "out of sync" in its menu and renders its badge as if in Unmanaged mode.
  • Use the config export and import feature to backup or replicate config settings. Exported config is represented as base64-encoded JSON.
